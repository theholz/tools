# Trading System Design (Project-Specific)

## Context
Multi-agent trading platform with infrastructure integration.

## Reference Agents
- SkillArchitect: Designs skills for domain + infrastructure
- DataOrchestrator: Manages data flow through platform
- AssetResearchOrchestrator: Monitors assets, updates context

## Design Principles
1. First principles decomposition (Principles Framework)
2. Infrastructure mapping (n8n, Neo4j, Letta, Perplexity)
3. Composable, testable skills
4. Explicit dependencies and fallbacks
